🚀 كيفية البدء (خطوات سهلة جداً):
═══════════════════════════════════════════════════════════════

الخطوة 1️⃣: افتح ملف setup.html
─────────────────────────────────────
• انقر مرتين على setup.html
• أو افتح به في المتصفح مباشرة
• سيشرح كل شيء خطوة بخطوة

الخطوة 2️⃣: إعداد Firebase (15 دقيقة)
─────────────────────────────────────
1. اذهب إلى https://console.firebase.google.com
2. أنشئ مشروع جديد باسم "احتواء"
3. فعّل: Authentication (Email/Password)
4. فعّل: Firestore Database (وضع التطوير)
5. انسخ بيانات الاعتماد

الخطوة 3️⃣: تحديث firebase-config.js
──────────────────────────────────────
1. افتح ملف: firebase-config.js
2. استبدل القيم "YOUR_..." ببيانات Firebase الفعلية
3. مثال:
   const firebaseConfig = {
     apiKey: "AIzaSyD...",  ← ضع القيمة الفعلية
     authDomain: "ehtewaa.firebaseapp.com",
     projectId: "ehtewaa",
     ... إلخ
   };

الخطوة 4️⃣: تشغيل المشروع (اختر واحدة):
  ────────────────────────────────────────

  ✅ الطريقة 1 (الأفضل - موصى بها):
     • ثبّت Live Server في VS Code
     • انقر كليك يمين على index.html
     • اختر: "Open with Live Server"
     • المتصفح سيفتح تلقائياً ✨

  ✅ الطريقة 2 (البسيطة):
     • انقر مرتين على index.html
     • سيفتح في المتصفح الافتراضي

  ✅ الطريقة 3 (احترافية - تحتاج Node.js أو Python):
     • ثبّت Node.js أو Python
     • افتح Terminal في المجلد
     • شغّل: npm start (أو python server.js)

الخطوة 5️⃣: اختبر التطبيق
─────────────────────────
1. اضغط "إنشاء حساب"
2. أدخل بيانات تجريبية:
   - الاسم: "أختبار"
   - البريد: "test@example.com"
   - الهاتف: "01000000000"
   - الحالة: "حامل"
   - كلمة المرور: "Test@123"
3. اضغط "إنشاء الحساب"
4. سيتم إعادة التوجيه للدخول
5. سجّل الدخول ببنفس البيانات
6. انقر على "لوحة التحكم"
7. تحقق من ظهور البيانات! ✅
