🎯 تطبيق احتواء - دليل البدء السريع
=============================================

✅ تم إعداد المشروع بنجاح!

📁 الملفات المضافة:
  • dashboard.html      - لوحة التحكم والإحصائيات
  • server.js           - خادم محلي
  • package.json        - معلومات المشروع
  • setup.html          - دليل الإعداد التفصيلي
  • README.md           - وثائق كاملة

🚀 البدء السريع:
================

1️⃣  إعداد Firebase:
   - اذهب إلى: https://console.firebase.google.com
   - أنشئ مشروع جديد أو استخدم موجود
   - تفعيل: Authentication (Email/Password)
   - تفعيل: Firestore Database (وضع التطوير)
   - انسخ بيانات اعتماد المشروع

2️⃣  تحديث firebase-config.js:
   - افتح ملف: firebase-config.js
   - استبدل القيم المكتوب فيها "YOUR_..."
   - مثال:
     apiKey: "YOUR_API_KEY" → apiKey: "AIzaSDWGSwaDG8w..."

3️⃣  تشغيل الخادم:
   - افتح PowerShell أو Command Prompt
   - اذهب لمجلد المشروع:
     cd "New folder (7)"
   
   - شغل الخادم:
     npm start
   
   - لا تغلق النافذة! اترك الخادم يعمل

4️⃣  فتح التطبيق:
   - افتح متصفحك
   - اذهب إلى: http://localhost:3000

✨ الصفحات المتاحة:
===================
• http://localhost:3000           → الصفحة الرئيسية
• http://localhost:3000/login.html → تسجيل الدخول
• http://localhost:3000/signup.html → إنشاء حساب جديد
• http://localhost:3000/dashboard.html → لوحة التحكم
• http://localhost:3000/setup.html → دليل الإعداد

🧪 اختبر التطبيق:
================
1. اضغط "إنشاء حساب"
2. أدخل بيانات حقيقية (البريد يجب أن يكون صحيحاً)
3. اختر "حامل" أو "أم" من الخيارات
4. اضغط "إنشاء الحساب"
5. سيتم إعادة التوجيه إلى تسجيل الدخول
6. سجل الدخول ببيانات التسجيل
7. انتقل إلى لوحة التحكم لرؤية البيانات

📊 لوحة التحكم تعرض:
====================
✅ إجمالي المستخدمين
✅ عدد النساء الحوامل
✅ عدد الأمهات الجدد
✅ عدد التسجيلات اليوم
✅ قائمة المستخدمين المسجلين
✅ سجل النشاط الأخير
✅ جدول تفصيلي لجميع البيانات

⚠️ ملاحظات مهمة:
================
• تأكد من تفعيل Authentication في Firebase
• تأكد من تفعيل Firestore Database في Firebase
• لا تشرك مفاتيح Firebase مع أحد
• استخدم وضع التطوير في الاختبار فقط
• قبل الإطلاق النهائي، طبق قواعس أمان شاملة

❓ مشاكل شائعة:
================

المشروع لا يعمل؟
→ تحقق من وحدة التحكم (F12) بحثاً عن أخطاء
→ تأكد من بيانات Firebase صحيحة
→ تأكد من تفعيل الخدمات في Firebase

لا أرى البيانات في Dashboard؟
→ تأكد من إنشاء مستخدم جديد أولاً
→ تحقق من Firestore في Firebase Console
→ تأكد من إعطاء الصلاحيات الصحيحة

الخادم لا يبدأ؟
→ تأكد من تثبيت Node.js: node --version
→ قد تحتاج npm install (إذا لم تعمل القيم الافتراضية)
